# 🧠 NeuralMatch AI: Clinical Case Similarity Engine

[![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![Lucide](https://img.shields.io/badge/Lucide-Icons-pink?style=for-the-badge)](https://lucide.dev/)
[![Recharts](https://img.shields.io/badge/Recharts-Data_Viz-orange?style=for-the-badge)](https://recharts.org/)

**Empowering medical professionals with AI-driven inference.** NeuralMatch AI compares real-time patient charts against global oncology and infectious disease datasets (including Mucormycosis) to accelerate diagnosis and clinical decision-making.

---

## ✨ Key Features

- 🔍 **Vector-Based Similarity Analysis**: Uses advanced pattern recognition to match patient symptoms against 14.2M+ historical records.
- 📊 **Dynamic Data Visualization**: Interactive Radar and Bar charts providing a multi-dimensional view of clinical matches.
- 🏥 **Clinical Cohort Matching**: Access anonymized patient profiles with the highest similarity scores for comparative study.
- 💎 **Premium Medical UI**: A high-end, glassmorphic dashboard designed for clarity in high-pressure clinical environments.
- 🧪 **Deep Inference Engine**: Real-time symptom clustering and oncology pattern recognition.
- 📑 **Comprehensive Reports**: Detailed treatment protocols and prognosis based on top clinical matches.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ai-similarity-system.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd ai-similarity-system
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Visit `http://localhost:5173` to view the application.
